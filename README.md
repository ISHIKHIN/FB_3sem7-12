## API Эндпоинты

### Аутентификация

| Метод | Эндпоинт | Описание | Доступ |
|-------|----------|----------|--------|
| POST | `/api/auth/register` | Регистрация пользователя | Гость |
| POST | `/api/auth/login` | Вход в систему | Гость |
| POST | `/api/auth/refresh` | Обновление токенов | Гость |
| GET | `/api/auth/me` | Получение текущего пользователя | Пользователь+ |

### Пользователи (только админ)

| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| GET | `/api/users` | Получить список пользователей |
| GET | `/api/users/:id` | Получить пользователя по id |
| PUT | `/api/users/:id` | Обновить информацию пользователя |
| DELETE | `/api/users/:id` | Заблокировать пользователя |

### Товары

| Метод | Эндпоинт | Описание | Доступ |
|-------|----------|----------|--------|
| GET | `/api/products` | Получить список товаров | Пользователь+ |
| GET | `/api/products/:id` | Получить товар по id | Пользователь+ |
| POST | `/api/products` | Создать товар | Продавец, Админ |
| PUT | `/api/products/:id` | Обновить товар | Продавец, Админ |
| DELETE | `/api/products/:id` | Удалить товар | Админ |

# Приложение магазин игрушек

## Технологии

### Бэкенд
- **Node.js** - среда выполнения JavaScript
- **Express.js** - веб-фреймворк
- **JWT (jsonwebtoken)** - аутентификация и авторизация
- **bcrypt** - хеширование паролей
- **Swagger UI** - документация API

### Фронтенд
- **React 18** - библиотека для построения интерфейсов
- **React Router v6** - маршрутизация
- **Axios** - HTTP клиент с перехватчиками
- **Context API** - управление состоянием аутентификации

## Функциональность

### Аутентификация и авторизация
- Регистрация новых пользователей
- Вход в систему с получением JWT токенов
- Автоматическое обновление access-токена через refresh-токен
- Хранение токенов в localStorage
- Защищенные маршруты на клиенте и сервере

### Управление товарами
- Просмотр списка товаров с фильтрацией по категориям
- Поиск товаров по названию и описанию
- Просмотр детальной информации о товаре
- Создание новых товаров (для продавцов и админов)
- Редактирование товаров (для продавцов и админов)
- Удаление товаров (только для админов)

### Управление пользователями (только для админов)
- Просмотр списка всех пользователей
- Изменение ролей пользователей
- Блокировка пользователей

## Роли и права доступа

| Роль | Доступные действия |
|------|-------------------|
| **Гость** (не аутентифицирован) | Регистрация, вход в систему |
| **Пользователь** (user) | Просмотр товаров |
| **Продавец** (seller) | Просмотр, создание, редактирование товаров |
| **Администратор** (admin) | Полный доступ: управление товарами и пользователями, изменение ролей |

## Установка и запуск

```bash
git clone <https://github.com/kirill944/FBD_7-12>
```

### Запуск бэкенда
```bash
cd backend
npm install
npm run dev
```
### Запуск фронтенда
```bash
cd frontend
npm install
npm start
```

Фронтенд будет доступен на http://localhost:3001 Бэкенд будет доступен на http://localhost:3000/api

## API Эндпоинты

### Аутентификация

| Метод | Эндпоинт | Описание | Доступ |
|-------|----------|----------|--------|
| POST | `/api/auth/register` | Регистрация пользователя | Гость |
| POST | `/api/auth/login` | Вход в систему | Гость |
| POST | `/api/auth/refresh` | Обновление токенов | Гость |
| GET | `/api/auth/me` | Получение текущего пользователя | Пользователь+ |

### Пользователи (только админ)

| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| GET | `/api/users` | Получить список пользователей |
| GET | `/api/users/:id` | Получить пользователя по id |
| PUT | `/api/users/:id` | Обновить информацию пользователя |
| DELETE | `/api/users/:id` | Заблокировать пользователя |

### Товары

| Метод | Эндпоинт | Описание | Доступ |
|-------|----------|----------|--------|
| GET | `/api/products` | Получить список товаров | Пользователь+ |
| GET | `/api/products/:id` | Получить товар по id | Пользователь+ |
| POST | `/api/products` | Создать товар | Продавец, Админ |
| PUT | `/api/products/:id` | Обновить товар | Продавец, Админ |
| DELETE | `/api/products/:id` | Удалить товар | Админ |
